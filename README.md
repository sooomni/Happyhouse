
# HappyHouse
### 공공데이터를 이용한 아파트 실거래가 제공 웹 사이트

### 📚 프로젝트 소개
* 개발 기간 : 2020.01 - 2020.05 (5개월)
* 사용 기술 : Spring boot, Servlet, JSP, Bootstrap, jQuery, Ajax, Vue,js
* 2인 팀 개발

### 🔨 주요 기능
1. 회원 관리 기능
	* 로그인/ 로그아웃
	* 회원정보 CRUD
	* 비밀번호 찾기
2. 게시판 기능
    * 공지사항 CRUD
    * Q&A 게시판 CRUD
3. 거래정보 관련 기능
    * 검색
        - 법정동, 아파트 명 검색
        - 개인 맞춤형 추천
        - 정렬 기능
    * 거래정보 조회 및 비교
        - 상세정보 비교
        - 실거래가 변동추이 조회
        - 비교 데이터 저장 및 열람
    * 지도 
        - 지도에 검색결과 마커 출력
        - 주변 편의시설 조회
